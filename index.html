DIVISI

Pemrograman Web

Di divisi ini akan mengajak semua anggota nya untuk membuat website, mulai dari desain UI sampai pemrograman. Target dari divisi ini adalah mampu menciptakan satu website portofolio untuk masing-masing peserta.

Pengembangan Game

Pengembangan game dan perangkat lunak merupakan divisi yang akan mengembangkan pemikiran kritis kepada para anggota. Melibatkan algoritma untuk menjalankan suatu program agar berjalan semestinya sesuai dengan logika komputer.

Robotik

Divisi dimana ekskul ini mulai berkembang. Dengan menggunakan Arduino UNO sebagai media pembelajaran hardware, mengizinkan para peserta untuk mengembangkan kreatifitas nya dalam menciptakan blue-print dan merancang komponen yang diperlukan.

Pengembangan Projek

Project Line Follower

Memanfaatkan motor shield L293D, 5 sensor inframerah, dua motor, dan yang terutama Arduino UNO. Robot ini akan berjalan mengikuti lintasan garis hitam yang sudah disediakan dan akan berhenti apabila garis hitam terdeteksi pada seluruh sensor.

Website Portofolio

Pembuatan website ini murni menggunakan HTML dan CSS. Pengembangan website ini dilakukan secara kolektif dan sudah responsive pada berbagai perangkat seperti handphone, tablet, dan laptop atau personal computer.
